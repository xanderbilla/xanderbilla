3. sleep Use to hold the terminal for the specified amount of time. By default it takes time in seconds.

Synatx: sleep NUMBER[s | m | h]

Example:

                            
billa@PROFESSOR:~$ man sleep
billa@PROFESSOR:~$ sleep 10s

6. mkdir Create the DIRECTORY(ies), if they do not already exist.

Synatx: mkdir [OPTION]... DIRECTORY...

Example:

                            
billa@PROFESSOR:~$ mkdir Vikas
7. rmdir Remove the DIRECTORY(ies), only if they already exist and empty.

Synatx: rmdir [OPTION]... DIRECTORY...

Example:

                            
billa@PROFESSOR:~$ rmdir Vikas
8. cd The cd utility shall change the working directory of the current shell execution environment

Synatx: cd [DIRECTORY]

Example:

                            
billa@PROFESSOR:~$ cd Downloads/
billa@PROFESSOR:~/Downloads$
9. touch Update the access and modification times of each FILE to the current time.

Synatx: touch [OPTION]... FILE...

Example:

                            
billa@PROFESSOR:~$ touch Vikas
billa@PROFESSOR:~$ ls file1 -l
-rw-rw-r-- 1 billa billa 0 Jul  4 07:21 file1 --> Note the time stamp
billa@PROFESSOR:~$ touch file1 --> modified the time stamp using `touch` command  
billa@PROFESSOR:~$ ls file1 -l
-rw-rw-r-- 1 billa billa 0 Jul  4 07:22 file1 --> Note the time stamp again
10. cat concatenate files and print on the standard output

Synatx: cat [OPTION]... [FILE]...

Example:

                            
billa@PROFESSOR:~$ cat > Sample ---> The `>` is used to create file
This is Xander Billa.
And who are you?
^C
billa@PROFESSOR:~$ cat Sample 
This is Xander Billa.
And who are you?
billa@PROFESSOR:~$
11. cp Copy files/directories from one SOURCE to DEST, or multiple SOURCE(s) to DIRECTORY

Synatx: cp [OPTION]...SOURCE DEST

12. mv Move files/directories from one SOURCE to DEST, or also used for renaming file/directory

Synatx: mv [OPTION]...SOURCE DEST

Example:

            billa@PROFESSOR:~$ mv File1 /Dir1/Vikas
