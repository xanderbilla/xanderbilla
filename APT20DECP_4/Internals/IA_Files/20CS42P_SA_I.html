Synatx: wc [OPTION]... [FILE]...

Example:

billa@PROFESSOR:~$ wc -c Sample

billa@PROFESSOR:~$ wc -l Sample 
2 Sample
billa@PROFESSOR:~$ wc -w Sample 
8 Sample

2. Vi Editor with commands

- Vim is a contarction of Vi improved is nan ecxtended version of the vi editor.
- Vim implements a text based user interface to advance text editing, and is favored by many system administrators and
softwares engineers for its efficiency and ability to be extensively customized
- Vim also include useful features such as text completion, syntax highlighting, spel checking, and many more.
- The vim command invoke the vim editoe
- This command automatically redirects the user to vim when entered without file name as an arguments

Syntax: vim [file name]

If the file doesn't exist it will create a new one.

Vim Modes

Vim is a modal editor and its different modes decide the functionality of various keys

Modes Enable Users To
Insert Insert text by typing
- To enable Execute mode press i key
Execute - Execute commands within Editor
- To enable Execute mode press : [command] followed by Esc key
Command - Perform different editingactions using keystrokes
- To enable command mode press Esc key
Visual - Highlight or select text for copying, deleting, and so on
- To enable Execute mode press v key

Editing operations

Editing operators in command mode are powerful tools that can be used to manipulates text with simple keystrokes.

Editing Enable Users To
x Delete the character selected by the cursor
d Delete text
dd Delete the current line
p Paste text on line directly below the cursor
P Paste text on line directly above the cursor
/{text String} Search through the document for specific text
?{text string} Search backward through the document for specific text
y Copy text
yy Copy the line directly above the cursor
c{range of lines} Begin a change in the specified range
u Undo the latest chanegs
U Undo all changes in the current line
ZZ Write the file only if changes were maed, then quit the vim editor

3. Linux command related to process and management

- A process is an instance of running program
- A process is said to be born when the program starts executuion and remain alive as long as the program is active - The kernal is responsible for management of process
- The important attributes of a process are:

  1. Process ID: Each process is uniquely identified by a unique integer value called PID or process ID
  2. Parent Process ID (PID): Parent also have the process ID

ps command ps command is used to view current runing processs on the system. It also determines memory usage, CPU usage, User ID, comand name etc.

Synatx: ps [OPTION]

Example:

                        
billa@PROFESSOR:~$ ps
    PID TTY          TIME CMD
  89384 pts/0    00:00:01 bash
 109954 pts/0    00:00:00 ps.

ps command options

option Used To
a List all user-triggered processes
-e List all process
-l List processes using a long listing format
u List process along with user name and start time
r Exclude process that are not running currently
x Include process without a terminal
f Search backward through the document for specific text
T Exclude process that were started by any terminal other than the current one
-U Display the p
-p Begin a change in the specified range
-C Undo the latest chanegs
--tty Undo all changes in the current line
