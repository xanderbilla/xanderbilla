Exp6: Program to implement Bubble Sort

            
#**********************************************************************************#
# Program to implememnt Bubble Sort                                                #
#**********************************************************************************#

 
n = int(input("Enter a number of element: "))
l =[]
for i in range(n):
    x = int(input("Enter element: "))
    l.append(x)
print("Unsorted list: ", l)
 
for i in range(n - 1):
        for j in range(n - 1):
            if (l[j] > l [j + 1]):
                temp = l[j]
                l[j] = l[j + 1]
                l[j + 1] = temp
print("Sorted list:", l)
----------------------------------Output: ----------------------------- 
        
Unsorted list: [25, 23, 32, 85, 77, 45, 12]
Sorted list: [12, 23, 25, 32, 45, 77, 85]
